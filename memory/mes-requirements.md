# MES 系统 - 业务需求分析

> **项目名称：** 充电器制造 MES 系统
> **技术栈：** SpringBoot 3 + Vue3 + MySQL 8 + MyBatis-Plus + JJWT
> **创建日期：** 2026-06-13
> **版本：** v1.0

---

## 一、项目背景

### 1.1 业务场景
本项目面向**充电器制造行业**，覆盖从订单下达到产品入库的完整生产流程。
通过 MES 系统实现生产过程的数字化管理，替代传统纸质工单和 Excel 管理方式。

### 1.2 系统目标
- **生产透明化：** 实时掌握工单进度、产能情况
- **质量可追溯：** 从成品反向追溯到原材料批次、操作员、设备
- **物料精准管控：** 减少浪费、避免缺料停工
- **权限分级管理：** 不同角色看到不同功能，保证数据安全

## 二、用户角色与权限

| 角色 | 说明 | 核心权限 |
|------|------|---------|
| **超级管理员** | IT 运维人员 | 所有功能 + 系统配置 |

| **班组长** | 产线主管 | 工单下发、报工审核、物料审批 |

## 四、业务流程图

### 4.1 生产主流程

```
┌─────────┐    ┌─────────┐    ┌─────────┐    ┌─────────┐    ┌─────────┐
│  创建工单 │───→│  审批下发 │───→│  领料备料 │───→│  生产报工 │───→│  质检入库 │
└─────────┘    └─────────┘    └─────────┘    └─────────┘    └─────────┘
```

### 4.2 工单流转详情

```
待发布 → 已下发 → 生产中 → 已完成
  │         │         │
  │    ┌────┘    ┌────┘
  │    ▼         ▼
┌┴────────┐ ┌──────────┐
│ 取消工单 │ │  暂停/返工  │
└─────────┘ └──────────┘
```

### 4.3 领料流程

```
操作工提交领料申请
        │
        ▼
班组长审批（审核 BOM 用量 vs 申请量）
        │
   ┌────┴────┐
   ▼         ▼
  通过      驳回
   │
   ▼
仓管员发料（扣减库存）
   │
   ▼
操作工确认收货 → 开始生产
```

### 4.4 质检流程

```
操作工报工 → 触发质检任务
                   │
                   ▼
质检员执行检验（按 QC 标准逐项检验）
                   │
              ┌────┴────┐
              ▼         ▼
             PASS      FAIL
              │         │
              ▼         ▼
           进入下工序   返工/报废
```

### 4.5 追溯流程

```
成品 SN 码扫描
    │
    ▼
查询该 SN 码的完整信息：
├── 哪个工单生产的？
├── 经过了哪些工位？
├── 哪些操作员操作过？
├── 用了哪些批次物料？
├── 质检结果如何？
└── 什么时候生产的？
```

## 五、功能模块详细需求

### 5.1 基础建模模块

#### 产品管理
- 新增/编辑/删除/查询产品
- 产品分类（快充/普通/无线）
- 产品编号自动生成或手动输入
- 产品状态管理（启用/停用）

#### 物料管理
- 新增/编辑/删除/查询物料
- 物料分类（芯片/外壳/线材/辅料）
- 库存数量管理 + 安全库存预警
- 物料编号自动生成

#### BOM 管理
- BOM 版本管理（支持多版本）
- BOM 明细：产品 → 物料 + 单件用量 + 损耗率
- BOM 状态流转：草稿 → 生效 → 作废
- 根据 BOM 自动计算工单用料清单

#### 工艺路线
- 定义产品生产工艺（如：焊接→组装→测试→包装）
- 工序排序（决定生产顺序）
- 每道工序绑定工位
- 标记哪些工序需要质检

#### 车间/产线/工位
- 车间 → 产线 → 工位 三级结构
- 工位类型（焊接/组装/测试/包装）
- 产线日产能配置

---

### 5.2 工单管理模块

#### 工单创建
- 手动创建 / 按 BOM 自动创建
- 选择产品 → 自动带出 BOM 和工艺路线
- 指定产线、车间、计划时间
- 工单编号自动生成（规则：MO + 日期 + 序号）

#### 工单下发
- 班组长审核工单后下发
- 下发后操作工可看到该工单
- 支持批量下发

#### 生产报工
- 操作工按工序报工
- 记录：报工数量、合格数、不良数、不良原因
- 支持按班次报工汇总
- 报工后自动更新工单完成进度

#### 工单管理
- 工单列表（搜索/筛选/分页）
- 工单状态跟踪（待发布/生产中/已完成/已取消）
- 工单暂停/恢复/取消
- 工单完成自动结算

---

### 5.3 物料管理模块

#### BOM 领料
- 按工单 BOM 自动生成领料清单
- 操作工/班组长提交领料申请
- 指定领料仓库

#### 领料审批
- 班组长审批领料申请
- 审批通过后通知仓管员

#### 发料管理
- 仓管员确认发料数量
- 库存扣减（实发数量）
- 发料后更新领料单状态

#### 退料管理
- 生产完成后剩余物料退回仓库
- 记录退料原因（多领/错领/剩余）
- 退回后库存增加

#### 库存管理
- 实时库存查询
- 安全库存预警（低于下限标红）
- 库存变动记录

---

### 5.4 质量管理模块

#### QC 标准
- 按产品+工序配置检验标准
- 定义检验项目（电压/外观/尺寸）
- 设定合格范围（上限/下限）
- 支持文本判定标准

#### 质检执行
- 触发方式：报工后自动生成质检任务
- 按标准逐项检验
- 实测值录入 + 自动判定 PASS/FAIL
- 支持批量检验

#### 质检记录
- 质检结果汇总（合格数/不合格数）
- 不良原因分类统计
- 质检报告导出

#### 产品追溯
- 通过 SN 码反向追溯：
  - 工单信息 → 工艺路线 → 经过工位
  - 操作员信息 → 质检记录
  - 物料批次信息 → BOM 用量
- 正向追溯：某批次物料用在了哪些成品上

---

### 5.5 设备管理模块

#### 设备台账
- 设备编号、名称、类型
- 所属车间/产线
- 设备状态（运行/停机/维修/报废）
- 制造商、购买日期

#### 保养管理
- 按日/月/年设定保养计划
- 保养记录（类型/内容/保养人）
- 下次保养日期提醒

#### 故障管理
- 故障报修（描述/等级）
- 维修记录（修复时间/维修人员）
- 故障统计（按设备/按类型）

---

### 5.6 权限管理模块

#### 用户管理
- 创建账号、分配角色
- 启用/禁用账号
- 修改密码
- 登录记录（时间/IP）

#### 角色管理
- 创建/编辑/删除角色
- 预设 6 类角色（管理员/经理/班组长/操作工/质检员/仓管员）
- 可自定义角色

#### 菜单权限
- 三级菜单结构（目录→菜单→按钮）
- 角色绑定菜单（控制可见性）
- 权限标识绑定（控制操作）

---

### 5.7 数据统计模块

#### 生产日报
- 按车间/产线/日期统计
- 计划产量 vs 完成产量 vs 合格产量
- 良品率计算

#### OEE 统计
- 设备综合效率（可用率 × 性能率 × 良品率）
- 按设备/产线/时间维度统计

#### 报表导出
- 工单报表、质检报表、物料报表
- 支持 Excel 导出

#### 看板（预留）
- 生产进度看板
- 不良品 TOP 排行
- 设备状态总览

## 六、非功能性需求

| 需求项 | 要求 |
|--------|------|
| **响应时间** | 接口响应 < 500ms，列表分页加载 < 1s |
| **并发量** | 支持 50+ 用户同时在线操作 |
| **数据安全** | 密码 BCrypt 加密，JWT Token 鉴权 |
| **可扩展性** | 模块化设计，后续可扩展 WMS/ERP 对接 |
| **可维护性** | 统一异常处理、统一返回格式、完整日志 |
